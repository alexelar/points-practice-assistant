# Exercise Assistant PWA

Simple voice-activated exercise assistant that works offline.

## Features
- Manual session start/stop
- Audio command playback
- Voice activity detection
- Offline support
- Installable PWA

## Usage
1. Click "Start Session" and allow microphone access
2. App will play command and listen for your voice
3. Cycle repeats: command → listen → command → listen
4. Click "Stop Session" when done
5. Change settings to suit your needs

## Commands
- Осмотри свое физическое тело
- Выбери точку на теле
- Прикоснись к этой точке
- Почувствуй пальцем прикосновение к точке
- Почувствуй точкой прикосновение пальца
- Определи температуру в точке
- Отпусти точку
